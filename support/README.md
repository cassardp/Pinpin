# Pinpin

**Private Moodboard & Bookmarks**

A local-first Pinterest alternative that respects your privacy. Save inspiration, organize your ideas, rediscover what matters. No ads. No algorithm. No account required.

## Features

- Save images & links from anywhere
- Pinterest-style grid layout
- Built-in OCR text search
- Visual search for similar products
- Offline storage + iCloud sync

## Perfect for

- Designers curating visual references
- Shoppers saving products to compare
- Travelers collecting places & itineraries
- Creatives building vision boards & mood boards
- Anyone looking for a private, ad-free bookmarking app

## Available On

- iOS 18.4+
- iPadOS 18.4+
- macOS 15+
